Development of Math 201

The new development is M201 is to address two goals: (1) The UN sustainability goals and (2) Bryant 2030’s vision skill and learning goals, which can be found here.

We believe that having the student working on a statistical project will satisfy these new requirements so we suggest the instructors to incorporate to their syllabus a project. We suggest that project should count for at least 10% of the total grades. We are also encouraged to use Excel for as much as possible, if not all, the statistical computations of the course.

This webpage contains examples and guideline on conducting a project for Math 201.

1. Project Structure

A statistical project is the process of answering a research question using statistical techniques and presenting the work in a written report. Although the final products of a typical statistical project likely to contains a written part (a report or an article) and a presentation part (slides or poster), the instructors should have the academic freedom to design their own project.
